# The Feed
Gather information with ease!

## Set Up
1. Run npm install

```
npm install
```

2. Run npm run dev

```
npm run dev
```

## Payment Feature
Test out using 4242 4242 4242 4242 as credit card number information. Other payment fields can be fabricated.

## Other Notes
Deployed heroku version currently does not have webhook features.
